post表单提交的参数的解码是在第一次调用request.getParameter时发生的，post表单的参数传递方式与QueryString不同，它是通过http的body传递
到服务端的。当我们再页面上单击提交按钮时浏览器首先将根据content-type的charset编码格式对在表单中填入的参数进行编码，然后提交到服务器端，在
服务器端同样也是用content-type的charset进行解码。所以通过post表单提交的参数一般不会出现问题，而且这个字符集是自己设置的，可以通过
request.setCharacterEncoding(charset)来设置。

你一定要在第一次调用request.getParameter方法之前就设置request.setCharacterEncoding(charset)。
